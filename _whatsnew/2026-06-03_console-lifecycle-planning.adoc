=== Lifecycle planning feature name updates 

The following Lifecycle planning features have name changes:

* The Tech refresh feature is now called *System Refresh*. Learn how to https://docs.netapp.com/us-en/console-lifecycle-planning/use/tech-refresh.html[evaluate a system refresh].
* The Capacity Additions feature is now called *Storage Expansion*. Learn how to https://docs.netapp.com/us-en/console-lifecycle-planning/use/capacity-add.html[expand storage].
* The Renewals feature is now called *Support Renewals*. Learn how to https://docs.netapp.com/us-en/console-lifecycle-planning/support/task-support-registration.html[register for support].
